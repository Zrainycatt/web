# 听澜书阁

一个功能完整、界面美观的听澜书阁平台原型。

## 项目特点

- 现代化设计，采用深蓝色主色调和暖橙色辅助色
- 完整的响应式布局，支持移动端、平板和桌面端
- 夜间模式切换功能
- 丰富的交互功能和用户体验优化

## 技术栈

- HTML5 语义化标签
- Tailwind CSS v3（响应式布局与样式）
- JavaScript（交互功能）
- Font Awesome（图标库）
- Google Fonts（Montserrat 和 Open Sans 字体）

## 页面结构

### 1. 首页 (index.html)
- 导航栏（Logo、菜单、搜索、夜间模式切换、登录按钮）
- 英雄区域（大标题、行动号召按钮）
- 热门分类（6个分类卡片）
- 热门推荐（5本书籍）
- 新书上架（5本书籍）
- 个性化推荐（5本书籍）
- 底部播放器栏
- 页脚

### 2. 分类页面 (category.html)
- 分类筛选器（全部、小说、历史、科学、商业、教育、艺术）
- 排序选项（热门、最新、评分、价格）
- 视图切换（网格/列表）
- 书籍列表（10本书籍）
- 分页控制
- 底部播放器栏

### 3. 书籍详情页 (detail.html)
- 书籍封面和基本信息
- 作者简介
- 内容简介
- 章节列表（32章）
- 用户评价系统（星级评分、评论列表）
- 主播信息
- 相关推荐
- 底部播放器栏

### 4. 播放页面 (player.html)
- 当前播放书籍信息
- 章节内容显示
- 进度条（可拖动）
- 播放控制（播放/暂停、上一章、下一章、快进、快退）
- 音量控制
- 倍速播放（0.5x - 2.0x）
- 定时关闭功能
- 章节列表
- 播放统计
- 快捷操作（书签、下载、笔记、分享）
- 相关推荐

### 5. 个人中心 (profile.html)
- 用户信息展示（头像、用户名、统计数据）
- 我的书架（收藏的书籍）
- 收听历史（带进度显示）
- 收藏列表
- 下载管理
- 会员中心（会员状态、特权展示）
- 收听统计（时长、偏好分析）

### 6. 搜索页面 (search.html)
- 搜索框（支持书名、作者、关键词）
- 搜索类型选择
- 热门搜索标签
- 搜索结果展示
- 相关推荐
- 底部播放器栏

### 7. 登录/注册页面 (login.html)
- 登录/注册切换
- 表单验证
- 社交登录（微信、QQ、微博）
- 记住我功能
- 忘记密码链接

## 核心功能

### 音频播放器
- 播放/暂停控制
- 进度条拖动
- 音量调节
- 倍速播放（0.5x、0.75x、1.0x、1.25x、1.5x、2.0x）
- 定时关闭（15分钟、30分钟、45分钟、1小时、1.5小时）
- 上一章/下一章切换
- 快进/快退（15秒）

### 书架管理
- 添加书籍到书架
- 从书架移除书籍
- 书架状态持久化（localStorage）
- 书架视图切换（网格/列表）

### 收藏功能
- 添加/取消收藏
- 收藏列表管理
- 收藏状态持久化

### 收听历史
- 自动记录收听历史
- 显示收听进度
- 历史记录持久化

### 搜索功能
- 多维度搜索（书名、作者、关键词）
- 热门搜索推荐
- 搜索结果排序

### 用户评价
- 星级评分（1-5星）
- 文字评论
- 评分统计展示

### 会员系统
- VIP会员状态
- 会员特权展示
- 会员续费功能

### 夜间模式
- 一键切换夜间模式
- 模式状态持久化
- 全页面适配

## 设计规范

### 色彩方案
- 主色调：深蓝色 (#1e3a8a)
- 辅助色：暖橙色 (#f97316)
- 中性色：白色、浅灰、深灰

### 排版规范
- 标题字体：Montserrat（无衬线字体）
- 正文字体：Open Sans
- 标题层级：H1-H4
- 行高：1.5-1.6

### 界面风格
- 简洁现代设计
- 卡片式布局
- 微妙阴影效果
- 线性风格图标
- 充足留白

## 响应式设计

- 移动端（< 640px）：单列布局
- 平板（640px - 1024px）：2-3列布局
- 桌面端（> 1024px）：5列布局

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可查看首页
2. 通过导航栏访问各个页面
3. 点击书籍卡片进入详情页
4. 在详情页点击"立即播放"或"加入书架"
5. 使用底部播放器控制播放
6. 在个人中心管理书架和查看历史
